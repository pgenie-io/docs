
Чтобы начать пользоваться инструментарием dbFirst вам необходимо получить доступ и установить CLI-приложение.

## Настройка доступа

Для того, чтобы получить доступ к платформе вам необходимо указать персональный токен доступа для приложения.

> Чтобы получить токен, если у вас его ещё нет, проследуйте на [dbFirst.ru](https://dbfirst.ru) и заполните форму доступа по программе Preview.

По пути `~/.dbfirst/config.yaml` создайте файл следующего содержания:

```yaml
token: ВАШ_ТОКЕН
```

## Установка приложения

Вы можете установить приложение как из дистрибутива для вашей платформы так и из исходников.

### Из дистрибутива

Скачайте дистрибутив для подходящей платформы:

- [Linux x64](https://dist.dbfirst.ru/dbfirst-cli-latest-linux-x64.tar.gz)
- [macOS x64](https://dist.dbfirst.ru/dbfirst-cli-latest-macos-x64.tar.gz)
- [Windows x64](https://dist.dbfirst.ru/dbfirst-cli-latest-windows-x64.zip)

Распакуйте в директорию, перечисленную в переменной окружения `PATH`. Например, в Linux или macOS это может быть `~/.local/bin`.

Если подходящего дистрибутива нет в списке, вы можете установить из исходников.

### Из исходников

Исходники дистрибутива публично доступны и сопровождаются инструкцией по установке. Для продолжения проследуйте в [Git-репозиторий](https://gitflic.ru/project/dbfirst/cli).
