# Support

---

## Community Support

For non-commercial support, questions, and bug reports, use the official GitHub channels:

- **[GitHub Discussions](https://github.com/pgenie-io/pgenie/discussions)** — Ask questions, share ideas, and discuss how to get the most out of pGenie. This is the best place for usage questions and feature requests.
- **[GitHub Issues](https://github.com/pgenie-io/pgenie/issues)** — Report bugs or request new features. Please include a minimal reproduction when reporting a bug.

Community support is provided on a best-effort basis.

---

## Commercial Support

Commercial support, consulting, and training for pGenie and related topics is available through [CodeMine](https://codemine.io).

CodeMine can help with:

- **pGenie setup and integration** — Getting pGenie running in your stack, configuring generators, integrating with your CI/CD pipeline.
- **Migrating existing codebases** — Strategies for incrementally adopting pGenie in a large codebase and development.
- **Custom code generator development** — Implementing generators for languages, frameworks, or internal tooling that are not covered by the existing open-source generators.
- **Database schema design and query optimization** — Designing schemas that work well with pGenie's type analysis, writing efficient parameterized queries, and tuning index coverage.
- **Training and workshops** — Onboarding your team on pGenie, PostgreSQL best practices, and type-safe database access patterns.

Contact [CodeMine](https://codemine.io) for more information.
