# Кодогенераторы

На данный момент поддерживается генерация кода для языков Java и Haskell.

Кодогенераторы версионируются с целью обеспечения обратной совместимости генерируемого кода при выпуске обновлений.

На данный момент доступны следующие кодогенераторы:

- [jdbc-v1](jdbc-v1.md)
- [hasql-v1](hasql-v1.md)
